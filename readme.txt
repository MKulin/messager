Компиляция и сборка:
1. Компиляция и сборки производятся при помощи механизма сборки Apache Maven.
    Скачайте и установите Apache Maven: http://www.apache-maven.ru/install.html
2. Из коммандной строки зайдите в директорию сохраненного приложения
   и выполните: mvn clean package

Запуск:
1. Для запуска экземпляра приложения "Издатель" в командной строке зайдите
   в, созданную после сборки, папку target и выполните:
   java -cp test.messager-1.0-SNAPSHOT.jar server.ServerMain port
   , где port - это номер порта, на котором хотите запустить приложение
2. Для запуска экземпляра приложения "Подписчик" в командной строке зайдите
   в, созданную после сборки, папку target и выполните:
   java -cp test.messager-1.0-SNAPSHOT.jar client.ClientMain host port
   , где host - адрес хоста ("Издателя")
         port - номер порта хоста ("Издателя")
